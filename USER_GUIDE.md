# 🎁 分享给朋友的简单指南

## 给没有编程背景的朋友

你好！这是一个UofT图书馆学习室预约查看器。

### 📦 如何安装（超简单）

1. **下载文件**
   - 你的朋友会给你一个文件夹或zip文件
   - 解压到桌面（如果是zip）

2. **安装应用**
   - 找到 `Install.command` 文件
   - 双击它
   - 输入密码（如果需要）
   - 完成！

3. **第一次运行**
   - 打开 "应用程序" 文件夹
   - 找到 "UofT Study Rooms" 
   - 双击打开

### ⚠️ 如果Mac说"无法打开"

别担心！这很正常：

1. 按住 `Ctrl` 键 + 点击应用
2. 选择 "打开"
3. 点击 "打开" 确认
4. 只需要第一次这样做

### 🚀 如何使用

1. **启动**
   - 双击应用图标
   - 等待浏览器自动打开

2. **获取数据**
   - 在左侧边栏点击 "🔄 Get Latest Data"
   - 等待几分钟（正在获取最新信息）

3. **查看房间**
   - 选择图书馆类型
   - 选择日期
   - 绿色=可预约，红色=已占用

4. **预约房间**
   - 点击绿色时间槽
   - 自动跳转到预约网站

### 🎯 小贴士

- **绿色块** = 可以预约的时间
- **红色块** = 已被预约
- **空白** = 没有数据或不开放
- 点击绿色块直接跳转预约！

### 🔧 如果遇到问题

1. **应用打不开**
   - 确保拖拽到了"应用程序"文件夹
   - 尝试右键 > 打开

2. **浏览器没打开**
   - 手动打开浏览器
   - 访问：http://localhost:8501

3. **没有数据**
   - 检查网络连接
   - 点击 "🔄 Get Latest Data"
   - 等待几分钟

4. **还是不行**
   - 联系给你这个应用的朋友
   - 或者重启电脑试试

### 📱 就这么简单！

不需要懂编程，不需要安装Python，直接用就行！
